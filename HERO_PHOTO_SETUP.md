# Hero 照片配置说明

## 添加你的照片

1. **准备照片文件**
   - 选择你想要展示的照片
   - 建议使用横向比例的照片（如 16:9 或 3:2）
   - 照片应该清晰、专业

2. **复制照片到项目**
   ```bash
   # 将你的照片复制到项目的 images 文件夹
   cp /path/to/your/photo.jpg /Users/helen/he-yun-personal-website/public/images/hero-photo.jpg
   ```

3. **或者使用命令行**
   如果照片在桌面上：
   ```bash
   cp ~/Desktop/你的照片.jpg /Users/helen/he-yun-personal-website/public/images/hero-photo.jpg
   ```

## 照片设计效果

现在的Hero区域包含了以下高大上的前端效果：

### 视觉效果
- ✅ **3D 视差效果**：鼠标移动时照片会有轻微的3D旋转
- ✅ **玻璃质感边框**：现代感的毛玻璃效果
- ✅ **渐变遮罩**：照片上有优雅的渐变过渡
- ✅ **悬浮粒子**：照片周围有动态的光晕效果
- ✅ **加载动画**：照片加载时显示优雅的加载状态

### 交互效果
- ✅ **鼠标跟随**：照片会根据鼠标位置进行3D旋转
- ✅ **悬停缩放**：鼠标悬停时照片会有轻微的放大效果
- ✅ **动态光效**：背景有呼吸般的光晕动画

### 装饰元素
- ✅ **右上角状态指示器**：带有脉冲动画的状态点
- ✅ **左下角身份标签**：玻璃质感的身份标识
- ✅ **悬浮粒子**：照片周围的装饰性光点

## 自定义效果

如果你想调整效果，可以编辑 `src/components/hero-image.tsx`：

- **调整旋转角度**：修改 `rotateY` 和 `rotateX` 的系数
- **调整动画速度**：修改 `transition` 的 duration
- **调整光效颜色**：修改 `bg-primary/10` 等颜色值
- **调整边框样式**：修改 `border` 和 `backdrop-blur` 参数

## 测试

添加照片后，刷新浏览器查看效果：
- 访问 http://localhost:3000
- 移动鼠标查看3D效果
- 悬停照片查看缩放效果

## 照片建议

- **风格**：专业、现代、有科技感
- **构图**：人物居中或略偏右，留有呼吸空间
- **背景**：简洁或与AI/科技相关
- **光线**：自然光或专业布光
- **表情**：自信、专业、友好
